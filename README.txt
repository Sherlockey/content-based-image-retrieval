- Instructions for running executables: Run all command line commands from project's root directory. Requires OpenCV. See .zed/tasks.json for the list of commands I used to build the project and run the separate tasks. Note, the task labeled "init" will be different on your machine. For example: "cmake -B build -S ." might be all you need to initialize. However, I had to point CMake to my installed vcpkg libraries to make CMake's search paths (via find_package()) locate my packages. I also use Ninja as the build generator. You can run any of the feature vector commands with any of the three distance metrics (sum-of-squared-distance, histogram-intersection, or cosine-distance); however, the distance metrics for baseline and combined-features are static. Change the file path to the target and the directory with images as needed.
- Instructions for running extensions: the extension tasks in .zed/tasks.json are labeled: "run orientation and color, cosine, 0535", and "run orientation and color, cosine, 0536". You can also use "run hsv histogram, histogram-intersection, 0164" to compare to the first color histogram in RGB. You can also run "run histogram, cosine-distance, 0164" to compare to the first color histogram which uses histogram-intersection.
